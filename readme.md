# Dynamic DNS Update Client (DUC) for Linux

Mantenha seu endereço IP atual sincronizado com seu host ou domínio No-IP com o  DUC - Dynamic Update Client (Cliente de Atualização Dinâmica) oficial do No-IP

# Must Change | Precisa alterar

Altere para as credenciais de acesso ao No-IP dentro do arquivo Dockerfile

ENV NOIP_USERNAME="CHANGEME"
ENV NOIP_PASSWORD="CHANGEME"
